Application mobile et site de gestion de contenu pour la British Human Computer Interaction 2014

Ce projet d'application mobile s'est vraiment divisé en deux partie, l'application sur le mobile et le site internet de gestion de contenu, en effet, les organisateurs peuvent rentrer les informations, les mettre à jour si nécessaire, et cela sans avoir à programmer quoi que ce soit, juste à remplir des formulaires. L'application est réalisée pour que l'utilisateur soit les informations les plus utiles en premiers.

L'application mobile est développée avec Apache Cordova, ce qui permet un développement multi-plateformes en utilisant seulement les outils du web comme le HTML5, le CSS3 et le JavaScript, pour l'application la librairie JavaScript est Angularjs.

Le site internet quant à lui est développé en utilisant le HTML5, le CSS3, le PHP et du JQuery comme librairie JavaScript. Il utilise une base de données MySQL et est développé en utilisant le patron MVC.

La liaison entre le site internet et l'application se fait en utilisant des fichiers au format Json.


Site internet pour une association

Le projet intitulé “Création d’un site internet pour une association” est un site web dynamique qui présente les différentes activités de la chorale Nadalenca. Le site comporte une session visiteurs qui permet d’avoir un aperçu des activités de l’association, et de contacter les membres. Il possède également une session membres pour consulter les partitions et les profils des autres membres, mais également une session administrateurs qui permet de gérer le contenu du site web.

Le site est développé en PHP, HTML5, CSS3, JavaScript. Il utilise une base de données MySQL. Il suit le patron Modèle-Vue-Contrôleur (MVC). Il sera par la suite hébergé sur OVH.
